# Changelog

## [2.0.0] - 2026-06-24

### Added
- 落款排版两种模式：公文落款（右空固定）和美观落款（最长行居中）
- 落款排版切换下拉控件（功能区 + 设置页同步）
- 元素管理面板（记忆管理），支持查看/删除/清空类型记忆
- 落款库自动记忆：排版后自动将落款文本存入落款库
- 标记元素功能：选择文本后标记为指定元素类型
- "X是"整句加粗功能

### Changed
- 功能区重构为四组布局，重新设计 12 个 SVG 图标
- BaseTxt 字符串抽象：检测/格式化基于指针集合而非段落遍历
- 微调面板重写：contenteditable + 逐字比对 + 灰色超出警告
- 插入落款改为始终打开面板，顶部加"自动插入时间"复选框
- 落款检测不再以句末标点结尾的行不因满行被拦截

### Fixed
- 标题整段被刷 h2 的 bug（格式化严格按 start/length 不扩段）
- 发言日期被 h3 抢检测的 bug
- 跨行标题续行缩进问题
- 落款/日期空格对齐算法
- 文档切换后面板不关闭问题
- 三级标题截断（truncateSegmentAtEndSymbol 传 rule.type）
- 日期模式匹配允许数字与汉字之间的可选空白
- 落款前补空行导致元素指针漂移

### Removed
- 旧式逐段扫描检测方式（完全替换为 BaseTxt 字符串）

## [1.0.0] - 2026-05-27

### Added
- 正式发布 1.0.0 版本
- 一键自动格式化公文
- 页面设置（A4 纸，37/35/28/26mm 边距）
- 字体设置（标题 2 号方正小标宋，正文 3 号仿宋 GB2312）
- 段落设置（固定行距 28.9 磅）
- 标题识别和格式化（一、二、三级标题）
- 页码设置（带横线，4 号宋体）
- 保存和恢复自定义设置
- 本地持久化存储设置
