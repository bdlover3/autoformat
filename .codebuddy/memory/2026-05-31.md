# 2026-05-31

## WPS加载项ribbon选项卡消失问题修复

### 问题
wpsjs build后上传，通过网页安装成功，WPS也提示安装了新加载项，但"公文排版"选项卡不显示。

### 根因
`App.vue`中`window.ribbon = ribbon`放在Vue的`onMounted`回调中设置，而WPS解析ribbon.xml后会立即调用`ribbon.OnAddinLoad`，此时Vue可能还未mount完成，导致`window.ribbon`未定义，回调失败。

### 修复
1. 将`window.ribbon = ribbon`移到`src/main.js`模块顶层立即设置
2. 从`App.vue`的`onMounted`中移除ribbon注册
3. 修复`docs/index.html`发布页版本号从1.0.0到1.0.1

### 关键代码位置
- `src/main.js`: window.ribbon注册（第6-10行）
- `src/components/ribbon.js`: ribbon回调函数定义
- `public/ribbon.xml`: ribbon UI定义，引用ribbon.OnAddinLoad等回调
- `docs/index.html`: 发布页面，包含版本号和7z下载URL
